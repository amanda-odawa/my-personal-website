## Installation
1. **Fork the repository**
2. **Clone the forked repository**
    ```bash
    git clone https://github.com/amanda-odawa/my-personal-website.git
    ```
3. **Navigate to the project directory**
    ```bash
    cd todo-list
    ```
4. **Open index.html in your web browser**
